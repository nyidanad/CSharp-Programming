# CSharp beadandó

### Parancsok
---
LIST
> [!NOTE]
> **ONLY ADMIN**
> Munkások adatainak kilistázása.
> **$ list**
> **$ ls**

ADD
> [!NOTE]
> **ONLY ADMIN**
> Munkás felvétele felhasználónév, jelszó alapján.
> **$ add {lname fname} {password}**
> **$ a {lname fname} {password}**

DELETE
> [!NOTE]
> **ONLY ADMIN**
> Munkás törlése a cégből.
> **$ delete {id}**
> **$ d {id}**

BALANCE
> [!NOTE]
> **ONLY ADMIN**
> Munkás fizetésének módosítása.
> **$ balance {id} {price}**
> **b {id} {price}**

CLEAR
> [!NOTE]
> **ONLY ADMIN**
> Konzol tisztítása.
> **clear**
> **c**

LOGOUT
> [!NOTE]
> **ONLY ADMIN**
> Kilépés az fiókból.
> **logout**
> **lo**
