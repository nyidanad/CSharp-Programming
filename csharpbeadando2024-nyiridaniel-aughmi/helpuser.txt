# buy
# ----------------------------------
  To purchase an item type the tool
  name and optionally the quantity.
  $ {tool} {?quantity}

# info
# ----------------------------------
  Information about a tool.
  $ {tool} info
  $ {tool} i

# storage
# ----------------------------------
  Opening your storage.
  $ storage
  $ st

# password
# ----------------------------------
  Changing your password.
  $ password
  $ pw

# logout
# ----------------------------------
  Logging out from your account.
  $ logout
  $ lo
